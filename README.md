# FlamesGame
A python game of the FLAMES game i learned in elementary school. This version has color support, and a "Simplified Mode" for  screen reader accessibility (trying out smth new)

⋆.𐙚 ̊  What does it have?
- Animated counting effect to reveal results !!

- Uses the library colorama for a very colorful game

- A "Simple" mode that removes aesthetic symbols for better compatibility with screen readers!

⋆.𐙚 ̊  Prerequisites
To run this game, you will need Python 3 installed on your machine. You will also need the colorama library.

⋆.𐙚 ̊  Installation

Clone this repo:

    git clone https://github.com/Nasvyni/flames-game.git
Navigate to the project folder:

    cd flames-game
Install the required dependencies:

    pip install -r requirements.txt
⋆.𐙚 ̊   How to Play
Run the script:

    python main.py
    
When prompted, type y if you want to use the Simplified Mode, or n for the Visual Mode.

Enter the two names you would like to find the compatibility of

see as the program processes the letters and reveals your compatibility result !!!!
